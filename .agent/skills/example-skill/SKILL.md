# Example Skill

This is a template skill demonstrating the structure.

## When to Use
Use this skill when you want to understand how skills work or as a starting point for your own skills.

## Instructions
1. Create a new folder in `.agent/skills/` or `~/.gemini/antigravity/skills/`
2. Add a `SKILL.md` file with YAML frontmatter
3. Define clear instructions for the agent

## Best Practices
- Keep skills focused on a single task
- Write clear descriptions so the agent knows when to use the skill
- Include example usage if helpful
